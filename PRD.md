## 뉴스 수집 · 요약 · 핵심 키워드 추출 앱

## 1. 제품 개요 (Product Overview)
### 1.1 제품명
News Insight Extractor (가칭)

### 1.2 목적
온라인 뉴스 사이트에서 뉴스를 자동 수집하고,
각 뉴스에 대해:

* 본문 추출
* 자동 요약
* 핵심 키워드 추출
  을 수행하여, 사용자가 짧은 시간 안에 이슈를 파악할 수 있도록 지원한다.

### 1.3 타겟 사용자
* 투자자, 트레이더
* 기획자, 마케터
* 리서처, 학생
* 뉴스 요약이 필요한 일반 사용자

### 1.4 문제 정의
* 뉴스 소비에 시간이 과도하게 소요됨
* 핵심만 빠르게 파악하기 어려움
* 키워드 기반 트렌드 파악이 수작업에 의존

### 1.5 해결 방향
* 뉴스 링크 자동 수집
* 본문 자동 파싱
* LLM 기반 요약
* NLP 기반 키워드 추출
* 결과를 구조화하여 제공


## 2. 핵심 기능 (Core Features)
### 2.1 뉴스 링크 수집
* 포털 메인(예: Daum)에서 주요 뉴스 링크 수집
* 카테고리별 수집 지원
* 정치, 경제, 사회, IT, 증권 등
* 사용자 직접 URL 입력도 가능

### 2.2 뉴스 본문 추출
* 라이브러리 기반 자동 파싱 (예: newspaper3k)
* 제목, 본문, 작성일, 언론사 추출
* 실패 시 fallback 파서 적용

### 2.3 뉴스 요약
* LLM 기반 요약 (OpenAI API 사용)
* 옵션:
  * 3줄 요약
  * 5줄 요약
  * 500자 이내 요약
* 제목 + 요약 함께 제공

### 2.4 핵심 키워드 추출
* 방식:
  * LLM 기반 키워드 추출
  * 또는 TF-IDF, TextRank 혼합
* 출력:
  * 상위 5~10개 키워드
  * 키워드 빈도/중요도 점수 포함

### 2.5 결과 관리
* 뉴스별 결과 저장
* 날짜, 카테고리, 키워드 기준 필터링
* CSV/Excel 다운로드


## 3. 사용자 시나리오 (User Flow)
1. 사용자가 앱 접속
2. 뉴스 수집 방식 선택
   * “오늘의 주요 뉴스 수집”
   * “특정 URL 입력”
3. 뉴스 수집 → 본문 파싱
4. 자동 요약 실행
5. 키워드 자동 추출
6. 결과 화면에:
   * 제목
   * 원문 링크
   * 요약
   * 키워드 태그
7. 사용자는:
   * 필터링
   * 저장
   * 파일로 다운로드

## 4. 기능 상세 정의 (Functional Requirements)
### 4.1 뉴스 수집
| 항목    | 설명                 |
| ----- | ------------------ |
| 입력    | 포털 자동 수집 / 사용자 URL |
| 출력    | 뉴스 URL 리스트         |
| 실패 처리 | 수집 실패 시 로그 기록 및 알림 |

### 4.2 본문 파싱
| 항목 | 설명                        |
| -- | ------------------------- |
| 파서 | newspaper3k 기본            |
| 결과 | title, text, source, date |
| 예외 | 파싱 실패 시 BeautifulSoup 백업  |

### 4.3 요약 기능
| 항목 | 설명       |
| -- | -------- |
| 엔진 | LLM API  |
| 옵션 | 요약 길이 선택 |
| 형식 | 제목 + 요약문 |

### 4.4 키워드 추출
| 항목 | 설명           |
| -- | ------------ |
| 방식 | LLM 또는 통계 기반 |
| 개수 | 5~10개        |
| 출력 | 키워드 + 중요도    |

### 4.5 결과 저장
| 항목    | 설명            |
| ----- | ------------- |
| 저장 방식 | 로컬 DB 또는 파일   |
| 검색    | 날짜, 키워드, 카테고리 |
| 내보내기  | CSV, Excel    |
